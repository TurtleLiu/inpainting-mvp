## 🧩 单元测试设计介绍

### 🔍 核心功能测试

#### 1. `test_prepare_mask_binary`
- **设计目标**: 验证掩码预处理功能，确保输出为二值图像
- **预期输入**: 64x64的灰度图像，中心区域为白色
- **预期输出**: 二值图像，像素值仅包含0或255

#### 2. `test_inpaint_returns_image`
- **设计目标**: 验证图像修复功能的正确性
- **预期输入**: 64x64的彩色图像和对应的掩码
- **预期输出**: 修复后的图像，尺寸与输入相同，包含正确的元数据

#### 3. `test_service_run_success`
- **设计目标**: 验证服务层的正常运行
- **预期输入**: 图像、掩码、算法参数
- **预期输出**: 成功返回修复结果

#### 4. `test_service_invalid_algorithm`
- **设计目标**: 验证服务层的参数校验
- **预期输入**: 无效的算法名称
- **预期输出**: 抛出ValueError异常

#### 5. `test_service_invalid_radius`
- **设计目标**: 验证服务层的参数范围校验
- **预期输入**: 无效的半径值(0)
- **预期输出**: 抛出ValueError异常
